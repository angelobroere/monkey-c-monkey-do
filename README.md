# 🐒 monkey-c-monkey-do

A practice Garmin watch face built while learning Connect IQ and Monkey C development. Nothing serious — just a playground for figuring things out.

## What it does

- Displays the current time and date
- Shows battery level, with a color warning when low
- Animated charging indicator
- Dynamic sky and ground background that changes color based on time of day
- A sun that rises and sets across the screen in a realistic arc

## Built with

- [Connect IQ SDK](https://developer.garmin.com/connect-iq/sdk/)
- Monkey C
- VS Code + Garmin Monkey C extension
- A lot of trial and error

## Target device

Garmin Forerunner 255

## Getting started

1. Install the [Connect IQ SDK](https://developer.garmin.com/connect-iq/sdk/) and VS Code with the Monkey C extension
2. Clone this repo
3. Open in VS Code and press `F5` to run in the simulator
4. Or deploy to a watch with developer mode enabled over WiFi
